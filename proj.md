# משימת כיתה – פרויקט REST API עם מודל רגרסיה לינארית

## 🎯 מטרת הפרויקט

בניית מערכת מלאה שמשלבת:

* מודל למידת מכונה (רגרסיה לינארית)
* REST API
* ניהול משתמשים והרשאות
* אבטחה באמצעות JWT

---

## 📌 תיאור כללי

בפרויקט זה תבנו מערכת שבה:

* משתמשים יכולים להירשם ולהתחבר
* רק משתמשים מחוברים יכולים לאמן מודל
* המודל חוזה ציון לפי שעות לימוד
* לכל משתמש נשמר מודל אישי

---

## 🧠 חלק 1 – מודל למידת מכונה

יש לממש פונקציות נפרדות:

### TRAIN

* אימון מודל רגרסיה לינארית
* פיצ'ר: מספר שעות לימוד
* תוצאה: ציון במבחן
* שמירת המודל לקובץ
* שם הקובץ = שם המשתמש

### PREDICT

* קבלת מספר שעות לימוד
* החזרת ציון צפוי
* שימוש במודל השמור של המשתמש

---

## 🌐 חלק 2 – REST API

### USERS API

* יצירת משתמש חדש
* שמירת נתוני משתמש

### AUTH LOGIN

* התחברות משתמש
* החזרת JWT Token

---

## 🔐 הרשאות ואבטחה

* רק משתמש עם JWT Token יכול:

  * להפעיל TRAIN
  * להפעיל PREDICT

---

## ⚙️ חלק 3 – ENDPOINTS נדרשים

### יצירת משתמש

POST /users

### התחברות

POST /login

### אימון מודל

POST /train
(דורש Token)

### חיזוי

GET /predict/{hours}
(דורש Token)
